# Replace non-Nigerian model images with verified African fashion photos (v2)

This change replaces 10 Pexels image IDs in `src/data/designs.ts` and 1 in `src/components/About.tsx` to correct images that previously depicted non-Nigerian/European models. The new IDs were verified to return HTTP 200 from the Pexels CDN. The diff is purely data: only numeric image constants change, with no structural or logic modifications to any component or utility.

**Watch for:** Visual content of replacement images cannot be verified from code review alone (confirmed) — the IDs resolve, but whether they depict Black/Nigerian models in culturally appropriate attire for each design title requires human spot-checking.

## High-level view

The change is entirely mechanical: 11 integer literals are swapped for 11 different integer literals. The `pexels()` helper, component rendering, and all surrounding logic remain untouched. The only runtime behavioral difference is which JPEG URLs the browser fetches. A wrong ID produces an inappropriate image but cannot break functionality.

The About section image moves from ID 3807517 to 3852974, following a user request for a designer/seamstress at a sewing machine with measuring tape. Whether the new ID better matches that description requires visual confirmation.

## Image-to-title correspondence

Ten designs spanning categories Ankara, Aso-Ebi & Owambe, Men's Tailoring, Bridal, and Corporate receive new image IDs. The code provides no mechanism to validate that a given Pexels photo matches the semantic content described in the `description` field. A reviewer should spot-check at least 3-4 of these URLs:

- `https://images.pexels.com/photos/8526759/pexels-photo-8526759.jpeg` — should show a man in a royal agbada (HF-015)
- `https://images.pexels.com/photos/33890252/pexels-photo-33890252.jpeg` — should show a lace bridal reception dress (HF-018)
- `https://images.pexels.com/photos/29076960/pexels-photo-29076960.jpeg` — should show dashiki corporate wear (HF-020)
- `https://images.pexels.com/photos/3852974/pexels-photo-3852974.jpeg` — should show a seamstress at a sewing machine (About section)

If any of these show unrelated content, the corresponding ID should be replaced before shipping.

## About section image update

This is the second iteration of that image selection. Visual confirmation that 3852974 matches the user's description is the only open verification step for this file.
